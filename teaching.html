Package R : STID

Il s'agit d'un package ayant vocation à être utilisé par les étudiants du BUT STID lors de leur formation, ainsi que de les suivre dans leur utilisation future de R. Pour installer et utiliser ce package, il vous faut :
  • Cloner le git depuis l'URL https://forge.clermont-universite.fr/git/stid_rpackage
  • Installer le package via le source
Ce package est en cours de construction, et il se peut donc que 1) le source en .tar.gz ne soit pas disponible (il faut donc construire le package) et 2) que les fonctions du package n'en soient pas à leur état final.
